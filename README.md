# proyecto-integracion-continua
proyecto-integracion-continua Universidad Politecnico Grancolombiano

El objetivo es implementar dos contenedores Docker que se comuniquen entre sí.

## 🚀 Servicios

- **API Node.js** en el puerto 3000.  
- **Base de datos MySQL** en el puerto 3306.  

## ▶️ Ejecución

1. Clonar el repositorio.  
2. Ejecutar `docker-compose up --build`.  
3. La API se conecta a la base de datos usando el host `db`.
